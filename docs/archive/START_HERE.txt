How to Use
==========

Using the Cache:

  let mut cache = create_fitness_cache();
  
  for individual in population {
      let tag = compute_phi_pi_fib_tag(&serialize(individual));
      
      let fitness = cache.get(tag)
          .unwrap_or_else(|| {
              let f = evaluate(individual);
              cache.insert(tag, f);
              f
          });
  }

Skip the Fibonacci search (use std::binary_search instead).

Performance Impact
==================

Cache: 2-5x speedup typical (depends on input repetition)
Search: -5% (actually makes things slower)
Net: Use cache only (skip search)

When to Use Cache:
  ✓ GA with 40%+ repeated evaluations
  ✓ Expensive fitness functions
  ✓ Transpilation caching
  ✓ Optimization pass caching

When to Skip:
  ❌ All unique fitness evaluations
  ❌ Trivial operations
  ❌ Already have other caching
